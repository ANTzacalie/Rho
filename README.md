This is the Rho Project

An open-source app built to store/share and scan all types of cards and NFC tags;
For now the app is supporting a basic password auth login and can store user registered credit card's only.

Future improvments:
  - App will support more type of cards storage;
  - Support for sharing custom app specific cards;
  - NFC cards sharing to others;
  - NFC tags support to scan/create;
  - QR Code and bar code camera scan functionality;
  - Link share of store cards;
